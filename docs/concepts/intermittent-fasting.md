# Intermittent Fasting

Skipping breakfast is "unnatural" by modern standards but **aligns with evolutionary patterns**. Our ancestors hunted/foraged before eating, using stored fat as fuel.

## Benefits

| Benefit | Mechanism |
|---------|-----------|
| Fat burning | Body switches to stored fat |
| Metabolic health | Improves insulin sensitivity |
| Reduced inflammation | Autophagy activation |
| Natural weight loss | Caloric deficit without restriction |

## Implementation

**Morning fasting approach:**
- Wake → hydrate (water, black coffee, tea)
- First meal 4-8 hours after waking
- Allows body to burn stored fat

**Caution:**
If you have **insulin resistance** (need carbs every meal, dizzy without food), start with **low-carb diet first**, then progress to time-restricted eating, then full morning fasting.

## Evolutionary Perspective

- Humans evolved fasting between meals
- Wildlife doesn't eat upon waking
- Modern "breakfast is essential" is marketing, not biology
