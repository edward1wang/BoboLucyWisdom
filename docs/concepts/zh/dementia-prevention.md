# 痴呆预防

阿尔茨海默病和认知衰退并非不可避免。多个可改变的风险因素。

## 循证策略

| 因素 | 行动 | 证据 |
|------|------|------|
| **咖啡** | 每日3杯 | 哈佛：43年研究，风险降低18% |
| **维生素D** | 维持50-80纳克/毫升 | 中年D水平预测后期风险 |
| **运动** | 每周150分钟中度运动 | 增加BDNF，血流量 |
| **睡眠** | 7-9小时 | 深睡期间清除淀粉样蛋白 |
| **社交** | 维持关系 | 哈佛80年研究 |
| **心脏健康** | 控制血压、胆固醇 | 与血管性痴呆关联 |
| **避免果糖** | 限制糖分 | 与阿尔茨海默病相关 |

## 早期征兆

- 忘记最近对话
- 找词困难
- 在熟悉地方迷路
- 性格变化
