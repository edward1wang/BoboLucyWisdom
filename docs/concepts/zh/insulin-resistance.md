# 胰岛素抵抗

细胞停止对胰岛素正常响应。血糖保持高位，胰岛素保持高位 → 代谢功能障碍。

## 症状

- 必须每几小时吃碳水
- 空腹时头晕/发抖
- 腹部脂肪（内脏脂肪）
- 餐后脑雾
- 皮赘、皮肤黑斑（黑棘皮病）

## 机制

1. 慢性碳水超载
2. 细胞"抵抗"胰岛素信号
3. 胰腺释放更多胰岛素
4. 高胰岛素 → 脂肪储存、炎症

## 逆转策略

| 方法 | 作用机制 |
|------|----------|
| **低碳水/生酮** | 去除葡萄糖负荷 |
| **间歇性禁食** | 降低胰岛素，恢复敏感性 |
| **运动** | 肌肉无需胰岛素即可吸收葡萄糖 |
| **睡眠7-9小时** | 睡眠不足恶化抵抗 |
| **减少果糖** | 果糖 → 肝脏胰岛素抵抗 |

## 警告

如果你有胰岛素抵抗，**不要直接跳到禁食**。先从低碳水开始，*然后*限时进食，*然后*再禁食。
