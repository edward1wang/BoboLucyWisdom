# 维生素K2

引导钙进入骨骼（而非动脉）的必需维生素。服用大剂量维生素D时的关键辅因子。

## 关键功能

**钙的交通控制器**：
- 激活骨钙素（将钙放入骨骼）
- 抑制血管钙化
- 预防动脉钙沉积

## 为何与维生素D同服

大剂量D增加钙吸收 → K2确保钙去骨骼，而非动脉。

## 食物来源

- 纳豆（发酵大豆）——含量最高
- 草饲黄油、酥油
- 蛋黄
- 硬质奶酪
- 肝脏

## 补充形式

| 形式 | 说明 |
|------|------|
| MK-4 | 合成，半衰期短 |
| MK-7 | 天然，半衰期长（首选） |

## 剂量

- 每日100-200微克MK-7
- 必须与维生素D3搭配
