# 二战不明飞行物（Foo Fighters）

二战空战期间（1940年代）出现的发光球体。轴心国和盟军飞行员都有报告。

## 特征

- 球形能量场
- 固体金属外壳（反射阳光）
- **跨维度能力：** 可穿过飞机
- 电磁干扰（干扰雷达、指南针）

## 官方回应

- **吉米·杜立特调查：** 向罗斯福报告为"星际飞行器"
- 立即引发国家安全担忧
- 未向公众提供解释

## 理论

| 理论 | 证据 |
|------|------|
| 德国秘密武器 | 双方都观察到 |
| 盟军秘密武器 | 同样问题 |
| 外星生命 | 穿越能力 |
| 自然现象 | 电磁效应不太可能 |

## 意义

现代军事背景下首个广泛记录的UFO现象。导致秘密调查项目的建立。
