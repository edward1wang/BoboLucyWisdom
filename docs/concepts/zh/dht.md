# DHT（二氢睾酮）

从睾酮衍生的雄性激素。男性型脱发和前列腺增生的主要驱动因素。

## 机制

- 5-α还原酶将睾酮 → DHT
- 效力是睾酮的3倍
- 结合毛囊雄激素受体 → 毛囊微型化

## 效应

| 有益 | 问题 |
|------|------|
| 体毛/面部毛发生长 | 头皮脱发 |
| 声音变低沉 | 前列腺增生 |
| 肌肉发育 | 痤疮 |
| 性欲 | 攻击性 |
