# 睾酮优化

无需药物干预即可维持健康睾酮水平的自然方法。

## 为何重要

低睾酮与以下问题相关：
- 慢性疲劳
- 性欲低下和勃起功能障碍
- 肌肉流失/脂肪堆积
- 抑郁和认知衰退
- 心血管疾病风险

## 天然促进剂

| 策略 | 机制 | 证据 |
|------|------|------|
| **睡眠7-9小时** | 睾酮合成在深睡高峰期 | 每晚5小时降低T 10-15% |
| **抗阻训练** | 急性和慢性睾酮升高 | 复合动作最佳 |
| **戒除种子油** | 减少omega-6炎症 | 种子油促炎 |
| **避免塑料/BPA** | 内分泌干扰 | BPA抑制LH |
| **日照10-20分钟** | 维生素D合成 | D是T前体 |
| **维生素D3** | 每日>4000国际单位，持续>12周 | 荟萃分析确认 |
| **锌30-60毫克** | 睾酮合成辅因子 | 仅短期使用 |
| **硼6-10毫克** | 降低性激素结合球蛋白 | 释放游离睾酮 |

## 降低睾酮的因素

- 睡眠不足
- 慢性压力（皮质醇）
- 体脂过多（芳香化）
- 酒精
- 内分泌干扰物（BPA、邻苯二甲酸盐）
- 过度训练
