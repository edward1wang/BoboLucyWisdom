# 罗斯威尔事件（1947年）

1947年7月：新墨西哥州罗斯威尔附近农场坠落物体。历史上最著名的UFO事件。

## 声称

- 飞碟坠毁
- 回收外星尸体
- 技术逆向工程
- 军方掩盖

## 官方说法（变化）

| 日期 | 解释 |
|------|------|
| 1947年 | "飞碟坠毁" — 最初报告 |
| 后来 | 气象气球 |
| 1994年 | 莫古尔间谍气球项目 |
| 1997年 | 测试假人（人形） |

## 为何是罗斯威尔？

- 1947年世界上唯一有核武器的地方
- 军事存在密集
- 偏远地点

## 逆向工程声称

**Philip Corso上校**（《罗斯威尔之后的日子》）：
- 来自飞碟的光纤
- 集成电路（IC）
- 夜视技术
- 凯夫拉
