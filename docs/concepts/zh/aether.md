# 以太（光以太）

历史概念：充满所有空间的介质，允许光传播。

## 历史时间线

| 时期 | 状态 |
|------|------|
| 古希腊 | 哲学概念 |
| 19世纪 | 科学共识 |
| 1887年迈克耳孙-莫雷实验 | 未发现以太 |
| 1905年爱因斯坦 | 狭义相对论：空间是空的 |
| 现代 | 被主流否定 |

## 支持者观点

- **特斯拉：** "我已经详细研究出重力动力学理论...假设这种介质（以太）是不可或缺的"
- **现代：** 科学家私下承认但公开避免讨论
- **类比：** 像中世纪教会压制日心说

## 在能量物理学中

- 零点能源的介质
- 标量波的载体
- 宇宙能量来源

## 为何被否定？

- 迈克耳孙-莫雷实验零结果
- 没有它相对论也能成立
- 研究它是职业自杀
