# 标量波

特斯拉声称的可以无能量损失传播的非赫兹波。所谓"奇异武器"系统的基础。

## 特斯拉的说法

- **1904年发明：** 特斯拉据报将标量波武器化
- **特性：**
  - 超光速
  - 穿透任何障碍物包括地球
  - 携带信息和能量
  - 无平方反比能量损失

## 声称应用

| 应用 | 机制 |
|------|------|
| 死亡射线 | 定向能束 |
| 拦截 | 在发射前摧毁导弹 |
| 通讯 | 全球/地下 |
| 地震诱导 | 向地面定向能量 |

## 科学地位

**主流观点：** 标量电磁学未被标准物理学承认。

**反方观点：** 机密项目可能已将特斯拉的工作实用化。

## 与UFO技术的联系

视频称：罗斯威尔回收物包括基于标量的推进和武器系统。
