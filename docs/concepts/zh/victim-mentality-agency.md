# 受害者心态与自主权

查理·芒格："每当你觉得某种情况或某个人正在毁掉你的生活时，实际上**是你在毁掉你的生活**。把自己看作受害者是毁掉你生活的灾难性方式。"

## 陷阱

| 受害者心态 | 自主权心态 |
|----------------|----------------|
| "市场导致我的投资失败" | "我做出的选择导致了这一结果" |
| 外部指责 | 内部责任 |
| 重复错误 | 学习并适应 |

## 受影响的领域

- 投资 → 责怪市场，重复错误
- 婚姻 → 责怪伴侣，无成长
- 职业 → 责怪经济/老板，停滞不前
- 人际关系 → 责怪他人，孤立

## 转变

**关键洞察：** 重拾自主权需要停止指责，承担责任——即使对"不是你的错"的事情。

> "重获生活控制权的路径是停止责怪他人，停止受害者心态，承担责任。"
