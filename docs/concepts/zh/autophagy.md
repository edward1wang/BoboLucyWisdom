# 自噬

细胞「自我吞噬」过程。细胞在营养匮乏期间回收受损成分。

## 机制

- 触发条件：禁食、运动、酮症
- mTOR（生长）↓ → 自噬（清理）↑
- 溶酶体降解并回收细胞碎片

## 益处

- 清除受损线粒体
- 清除错误折叠蛋白
- 可能延缓衰老
- 减少炎症

## 如何诱导

| 方法 | 所需时间 |
|------|----------|
| 水禁食 | 16-24+小时 |
| 生酮饮食 | 持续酮症 |
| 运动 | 规律训练 |
| 限制热量模拟物 | 补充剂（谨慎） |

## 与禁食的关系

间歇性禁食触发肝脏、肌肉和大脑的自噬。
