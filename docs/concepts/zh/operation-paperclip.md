# 回形针行动

美国政府1945-1959年实施的项目，招募了1600多名德国科学家、工程师和技术人员——包括纳粹党员——为美国政府工作。

## 视频中的关键说法

- **不是渗透** — 科学家用知识换取豁免
- 与苏联争夺德国技术人才
- 战后在美国发现的纳粹机库中找到外星飞行器

## 著名招募对象

| 科学家 | 纳粹角色 | 美国角色 |
|--------|----------|----------|
| **沃纳·冯·布劳恩** | V2火箭开发者 | NASA阿波罗架构师 |
| **胡贝图斯·施特鲁格霍尔德** | 空军研究员 | 航天医学先驱 |

## UFO关联

- 冯·布劳恩 allegedly 警告"假外星人入侵"是最后的假旗行动
- "航天之父" — SpaceX技术血统可追溯至V2火箭

## 争议

- 洗白纳粹历史
- 德国火箭项目使用犹太奴工
- 自1940年代以来火箭技术没有革命性改变
