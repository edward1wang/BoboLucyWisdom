# 冥想与专注力

**常见体验：** 静坐时，思绪纷飞、分心不断。这对未经训练的心灵来说是正常的。

## 解决方案：冥想练习

**目的：** 不是"没有念头"，而是对念头的**不执着**。

**比喻：**
- 心灵如空容器
- 念头自由来去
- 不阻挡，不抓取
- 念头自由流动，无阻碍

## 益处

| 未经训练 | 经过训练 |
|------------------|---------------|
| 被每个念头分心 | 观察念头而不纠缠 |
| 无法平静心灵 | 随时获得心智清明 |
| 注意力分散 | 深度专注 |
| 无法进入心流 | 心流状态触手可及 |

**心流状态就是一种冥想状态**——从空容器过渡到有序的房间。

## 实用入门

1. 任何冥想指导都有效
2. 从5分钟开始，逐渐延长
3. 一致性胜过时长
