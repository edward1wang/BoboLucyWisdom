# 长寿

延长健康寿命（健康期），而非仅仅增加年数。

## 关键支柱

| 支柱 | 行动 |
|------|------|
| **营养** | 低糖、充足蛋白质、健康脂肪 |
| **运动** | 抗阻 + 2区心肺 |
| **睡眠** | 7-9小时，规律作息 |
| **社交** | 牢固关系（哈佛80年研究） |
| **心智** | 持续学习、目标感 |
| **补充剂** | 维生素D、Omega-3、NAD+前体 |

## 追踪的生物标志物

- HbA1c（血糖控制）
- ApoB（心血管）
- 维生素D
- 炎症标志物（高敏C反应蛋白）
