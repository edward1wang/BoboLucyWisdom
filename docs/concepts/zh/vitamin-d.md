# 维生素D

关键激素（不只是维生素），参与大脑健康、免疫力、骨骼健康和长寿。全球约10亿人缺乏。

## 大脑健康（痴呆预防）

**16年研究发现：**
- 中年维生素D水平预测晚年痴呆风险
- D充足 = Tau蛋白堆积较少（阿尔茨海默病标志物）
- 机制：激活小胶质细胞（大脑的清理 crew）

## 癌症与免疫

安格斯·达格利什博士（肿瘤学家）：维生素D补充提高免疫治疗应答率。称之为"廉价、有效、无毒" vs 昂贵、有毒的药物。

## 最佳水平

| 状态 | 25(OH)D水平 |
|--------|---------------|
| 缺乏 | <20 ng/mL |
| 充足 | 30-50 ng/mL |
| 最佳 | 50-80 ng/mL |

## 补充方案

**如果缺乏：**
- 短期5000-10000国际单位D3（医疗监督）
- 维持期：每日2000-5000国际单位D3

**必须同时服用：**
- **维生素K2**（将钙导向骨骼，而非动脉）
- **镁**（D激活所需）
- 随高脂餐服用（脂溶性）

## 阳光 vs 补剂

**自然合成：**
- 紫外线B暴露触发皮肤产生
- 正午阳光15-30分钟 = 10000-20000国际单位
- 防晒霜阻挡紫外线B → 阻挡D合成

**策略：** 先晒20-30分钟无防护阳光，*然后*如有需要再涂防晒霜。
