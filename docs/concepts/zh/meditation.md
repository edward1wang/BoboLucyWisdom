# 冥想

不带执念地观察思绪的心理训练。通往专注注意力和减少反刍的通道。

## 核心练习

- **目标**：不是"无思绪"，而是**不执着**
- **技巧**：专注呼吸；分心时，轻轻拉回
- **进步**：从5分钟开始，逐渐延长

## 益处

- 减少压力（降低皮质醇）
- 提高专注和集中力
- 更好的情绪调节
- 进入西塔脑波
- 进入心流状态的通道

## 类型

| 类型 | 重点 |
|------|------|
| 正念冥想 | 开放觉察 |
| 专注冥想 | 单点（呼吸） |
| 瑜伽睡眠 | 深度放松 |
| 身体扫描 | 身体觉知 |
