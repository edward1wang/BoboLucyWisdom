# 间歇性禁食

跳过早餐按现代标准是"不自然"的，但**符合进化规律**。我们的祖先在进食前先狩猎/采集，使用储存的脂肪作为燃料。

## 益处

| 益处 | 机制 |
|---------|-----------|
| 脂肪燃烧 | 身体切换到储存脂肪 |
| 代谢健康 | 改善胰岛素敏感性 |
| 减少炎症 | 自噬激活 |
| 自然减重 | 热量缺口而非限制 |

## 实施方案

**晨间禁食方法：**
- 醒来 → 补水（水、黑咖啡、茶）
- 醒来后4-8小时吃第一餐
- 让身体燃烧储存的脂肪

**注意：**
如果你有**胰岛素抵抗**（每餐都需要碳水化合物，不吃东西就头晕），先从**低碳水饮食**开始，然后逐步过渡到限时进食，最后进行完整的晨间禁食。

## 进化视角

- 人类在餐间进化出禁食机制
- 野生动物不会在醒来后立即进食
- 现代"早餐必不可少"是营销，不是生物学
