# 慢性疲劳与微量营养素缺乏

大多数人认为疲劳是睡眠问题。通常不是。即使8小时睡眠，如果你缺乏关键微量营养素，也会感到疲惫。

## 四大缺乏症

| 营养素 | 检测 | 作用 |
|----------|------|------|
| **镁** | 血清/红细胞镁 | 能量产生、肌肉功能 |
| **维生素D** | 25(OH)D | 大脑健康、免疫力、能量 |
| **维生素B12** | 血清B12 | 神经功能、能量代谢 |
| **铁蛋白** | 血清铁蛋白 | 铁储存、氧气运输 |

## 检测原则

> "不要猜测。检测。一个简单的血液面板可以改变一切。"

—— 加里·布雷卡

## 为何缺乏被忽视

- 症状（疲劳、虚弱）非特异性
- 标准面板通常*不包括*这些
- "正常"范围基于人口平均值，而非最佳功能

## 行动步骤

1. 要求完整的微量营养素检测
2. 对照*最佳*（而非仅正常）水平检查
3. 根据结果针对性补充
4. 每3-6个月复测
