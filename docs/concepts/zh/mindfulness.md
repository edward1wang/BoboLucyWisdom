# 正念

不带评判的当下觉察。心理清晰和情绪调节的基础。

## 核心练习

- **注意**：专注于当下体验
- **态度**：不评判、好奇
- **觉察**：对任何出现的事物开放

## 益处

- 减少反刍
- 降低皮质醇（压力荷尔蒙）
- 提高专注力
- 更好的情绪调节
- 减轻焦虑

## 与冥想的区别

| 冥想 | 正念 |
|------|------|
| 正式练习 | 随时随地 |
| 特定技巧 | 存在方式 |
| 静坐练习 | 行走、进食、呼吸 |

## 实际应用

- 正念呼吸
- 身体觉察
- 正念饮食
- 行禅
