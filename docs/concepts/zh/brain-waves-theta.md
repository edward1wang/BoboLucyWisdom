# 脑电波与西塔波状态

五种脑电波频率对应不同的意识状态。西塔波（θ）是通往潜意识的"门户"，直觉和创造力在此 flourish。

## 五种脑电波频率

| 波段 | 频率 | 状态 |
|------|-----------|-------|
| 伽马波 (γ) | 30-100 Hz | 巅峰洞察、心流状态 |
| 贝塔波 (β) | 13-30 Hz | 主动思考、解决问题（默认工作模式） |
| 阿尔法波 (α) | 8-13 Hz | 放松的觉察 |
| **西塔波 (θ)** | **4-8 Hz** | **创造力、直觉、深度冥想** |
| 德尔塔波 (δ) | 0.5-4 Hz | 深度睡眠、愈合 |

## 西塔波状态益处

- 潜意识访问
- 创造性突破（许多发明诞生于此）
- 直觉觉知
- 深度放松

## 如何在白天诱导西塔波

1. **午后小憩** — 短暂休息触发西塔波
2. **瑜伽休息术** — 引导式深度放松
3. **双耳节拍** — 音频频率夹带
4. **感官剥夺** — 漂浮舱、眼罩、想象漂浮在虚空中
