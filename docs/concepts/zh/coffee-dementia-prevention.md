# 咖啡与痴呆预防

**哈佛大学43年研究（n=131,821）：** 晨间咖啡可降低痴呆风险18%。脱咖啡因咖啡**无效**。

## 机制：腺苷拮抗

- 腺苷在白天累积 → 导致困倦 + 抑制记忆神经递质
- 咖啡因在腺苷结合前阻断腺苷受体
- 记忆回路运行不受抑制

## 为何脱咖啡因咖啡无效

脱咖啡因咖啡含有：
- 相同的多酚
- 相同的抗氧化剂  
- 相同的镁

**缺失：** 阻断腺苷受体的分子。

如果多酚是机制，脱咖啡因咖啡应显示部分效果。它显示**零**效果。

## 剂量反应

- 约**3杯**时达到平台期
- 5杯 ≠ 优于3杯（只是紧张不安）
- 即使对高遗传风险者也有效

## 关键引言

> "咖啡是地球上消费最广泛的精神活性药物。每天20亿杯。我们花了数十年的时间把它当作一种轻微的恶习。结果发现这是人类历史上最大规模的意外痴呆预防项目。"
