# 奇异武器

军工复合体据称开发的超越常规战的先进机密武器。

## 类型

| 武器 | 描述 | 状态 |
|------|------|------|
| **死亡射线** | 定向能光束 | 特斯拉声称成功 |
| **标量波** | 非赫兹波，特斯拉概念 | 理论/争议 |
| **力场** | 能量屏障 | 科幻，无证据 |
| **天气改造** | 类似HAARP的电离层加热 | 有文献记载，有限 |
| **精神控制** | 电磁/无线电神经效应 | 研究阶段 |

## 特斯拉的死亡射线

- 声称能在200英里以上击落飞机
- 据称向美国军方演示过
- 从未公开验证

## 理论基础

- 特斯拉1930年代"远距传力"专利
- 标量电磁学
- 零点能源武器

## 怀疑论观点

- 非凡的主张需要非凡的证据
- 无特斯拉原型幸存
- 冷战从未部署"死亡射线"
