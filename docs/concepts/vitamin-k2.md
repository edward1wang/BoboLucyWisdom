# Vitamin K2

Essential vitamin that directs calcium to bones (not arteries). Critical cofactor when taking high-dose vitamin D.

## Key Function

**Calcium traffic controller**:
- Activates osteocalcin (puts calcium in bones)
- Inhibits vascular calcification
- Prevents arterial calcium deposits

## Why Take With Vitamin D

High-dose D increases calcium absorption → K2 ensures it goes to bones, not arteries.

## Food Sources

- Natto (fermented soybeans) — highest
- Grass-fed butter, ghee
- Egg yolks
- Hard cheeses
- Liver

## Supplementation

| Form | Notes |
|------|-------|
| MK-4 | Synthetic, shorter half-life |
| MK-7 | Natural, longer half-life (preferred) |

## Dosage

- 100-200 mcg MK-7 daily
- Always pair with vitamin D3
