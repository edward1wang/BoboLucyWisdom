# Longevity

Extending healthy lifespan (healthspan), not just chronological years.

## Key Pillars

| Pillar | Action |
|--------|--------|
| **Nutrition** | Low sugar, adequate protein, healthy fats |
| **Exercise** | Resistance + zone 2 cardio |
| **Sleep** | 7-9 hours, consistent schedule |
| **Social** | Strong relationships (Harvard 80-year study) |
| **Mental** | Continuous learning, purpose |
| **Supplements** | Vit D, Omega-3, NAD+ precursors |

## Biomarkers to Track

- HbA1c (glucose control)
- ApoB (cardiovascular)
- Vitamin D
- Inflammatory markers (hs-CRP)
