# Brain Waves and Theta State

Five brain wave frequencies correspond to different states of consciousness. Theta waves (θ) are the "gateway to subconscious" where intuition and creativity flourish.

## The Five Brain Wave Frequencies

| Wave | Frequency | State |
|------|-----------|-------|
| Gamma (γ) | 30-100 Hz | Peak insight, flow states |
| Beta (β) | 13-30 Hz | Active thinking, problem-solving (default work mode) |
| Alpha (α) | 8-13 Hz | Relaxed awareness |
| **Theta (θ)** | **4-8 Hz** | **Creativity, intuition, deep meditation** |
| Delta (δ) | 0.5-4 Hz | Deep sleep, healing |

## Theta State Benefits

- Subconscious access
- Creative breakthroughs (many inventions born here)
- Intuitive knowing
- Deep relaxation

## How to Induce Theta During Day

1. **Afternoon nap** — Short rest triggers theta
2. **Yoga Nidra** — Guided deep relaxation
3. **Binaural beats** — Audio frequency entrainment
4. **Sensory deprivation** — Float tank, eye mask, imagining floating in empty space
