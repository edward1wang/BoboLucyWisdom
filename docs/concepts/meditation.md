# Meditation

Mental training to observe thoughts without attachment. Gateway to focused attention and reduced rumination.

## Core Practice

- **Goal**: Not "no thoughts" but **non-attachment**
- **Technique**: Focus on breath; when distracted, gently return
- **Progress**: Start 5 min, extend gradually

## Benefits

- Reduced stress (lowers cortisol)
- Improved focus and concentration
- Better emotional regulation
- Access to theta brain waves
- Gateway to flow states

## Types

| Type | Focus |
|------|-------|
| Mindfulness | Open awareness |
| Concentration | Single point (breath) |
| Yoga Nidra | Deep relaxation |
| Body scan | Somatic awareness |
