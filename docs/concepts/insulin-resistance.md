# Insulin Resistance

Cells stop responding properly to insulin. Blood sugar stays high, insulin stays high → metabolic dysfunction.

## Symptoms

- Must eat carbs every few hours
- Dizzy/shaky when fasting
- Belly fat (visceral adiposity)
- Brain fog after meals
- Skin tags, dark skin patches (acanthosis)

## Mechanism

1. Chronic carb overload
2. Cells "resist" insulin signal
3. Pancreas releases MORE insulin
4. High insulin → fat storage, inflammation

## Reversal Strategies

| Method | How It Works |
|--------|--------------|
| **Low-carb/Keto** | Removes glucose load |
| **Intermittent fasting** | Lowers insulin, restores sensitivity |
| **Exercise** | Muscles soak up glucose without insulin |
| **Sleep 7-9h** | Sleep deprivation worsens resistance |
| **Reduce fructose** | Fructose → liver insulin resistance |

## Warning

If you have insulin resistance, **don't jump straight to fasting**. Start with low-carb, *then* add time-restricted eating, *then* fasting.
