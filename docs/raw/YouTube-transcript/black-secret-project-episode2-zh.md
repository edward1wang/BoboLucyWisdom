## UFO计划演变
- **1947年9月** — Project Sign (信号计划) + MJ-12成立

- **1952年** — Project Blue Book (蓝皮书计划) = 掩盖计划

## 重要文件
1. **1950年FBI给胡佛的简报** — 罗斯威尔事件，3架坠毁

3. **1953年CIA心理战备忘录** — 利用UFO进行欺骗行动

## 1956年关键事件
- Eisenhower组建PFIAB (后更名为PFIAC)
- Nelson Rockefeller领导重组五角大楼/CIA/DIA
- 创造了"迷宫"般的保密系统
- 总统和国会被系统性地隔绝

- **标量波武器** — 特斯拉死光，击落飞碟
