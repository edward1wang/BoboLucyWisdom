## Post

普通人也能重写自己的生物学——不用药、不用手术，就能成为“超自然”的自己！ 我并不是在谈论玄学。这时64岁的Dr. Joe Dispenza，在扫描了成千上万普通冥想者的脑部后，得出的惊人结论。它是20多年表观遗传学和神经科学的研究成果。 你为什么感觉生活一直卡住？因为你正困在一个隐形循环里： 想法 → 化学反应 → 情绪 → 记忆 → 回到相同想法…… 你每天都在无意识地“排练”这个循环。大脑分不清生动记忆和真实事件——你反复回想过去的痛苦，身体就释放压力激素，仿佛它正在发生；你一直担忧未来，也是一样。 同样的 routine = 同样的生活。大脑是预测机器，你每天喂它相同的想法、情绪和行为，它就只会预测并制造更多“相同”的明天。 怎样才能跳出这个循环？ 你需要放下现实自我ego，放下你的逻辑大脑，重新与你更深层的意识链接。 如何做到这一点？ 【一】启动高频情绪。 高频情绪是指的是那些振动频率较高、正向且具创造力的情绪，能帮助我们从“生存模式”切换到“创造模式”，从而重塑大脑、身体和现实。 简单来说： 高频情绪 = 感恩、爱、喜悦、欣赏、慈悲、敬畏、灵感、自由等。 低频情绪 = 恐惧、愤怒、悲伤、内疚、嫉妒、焦虑、羞耻等（这些频率较低、较慢，常把我们困在过去）。 高频情绪不是“假装开心”，而是真实地在内心生成这些感觉，不需要外部事件发生。你的身体和潜意识会把这些情绪当成“已经发生”，从而提前为新未来做好生化准备。 【二】通过冥想来重塑你的神经通路。 关键在于你冥想时的脑波状态，而不是时间长短： 我们大多数清醒的时间里都处于Beta波段：压力、生存模式。大多数人带着焦虑可视化。其实你一直都在显化，世界按照你焦虑的样子发展。 所以冥想的目的就需要让脑波从Beta波降到： Alpha波：平静专注 Theta波：深度放松、易受暗示 有意思的是，当你真正进入深度冥想，脑波会出现短暂的高频Gamma波，这是高度整合的“超自然”状态，也是通识打通的瞬间，狂喜的状态。 还有一个被忽略的秘密——心脏是第二个大脑。当心率达到“连贯”状态时，大脑会自动跟随。把手放在心口，缓慢深呼吸，感受真切的感恩、喜悦或爱，那一刻你就在创造改变的电磁场。 最实用的技巧之一是“Mind Movie”： 选几张代表你理想未来的图片，在冥想中像电影一样播放，同时强烈感受它已经实现的情绪。你的潜意识不知道它还没发生，它会开始为新现实构建身体和行为。 成为超自然的自己，其实就这几步： 觉察并中断旧循环 改变日常routine，引入新鲜输入： 用高频情绪强化清晰意图 进入正确脑波状态冥想 在心脏锚定感恩 每天排练未来，直到身体先于现实相信它 真正的转变，不是发生在头脑里，而是当你的身体开始相信新未来的那一刻。 ～关注我，一起探索心灵自由之路～ 波波 Lucy (@BoboLucyWisdom) x.com/i/article/203258958132… — https://nitter.net/BoboLucyWisdom/status/2035977617195835646`#m`
