# 尼古拉·特斯拉

塞尔维亚裔美国发明家、工程师和未来学家（1856-1943）。交流电系统的先驱。

## 关键贡献

- **交流电系统** — 全球标准
- **感应电机** — 无刷设计
- **特斯拉线圈** — 共振变压器
- **无线传输** — 沃登克里弗塔

## 自由能源研究

- **辐射能量** — 宇宙射线
- **无线电力** — 全球分配
- **放大发射器** — 自由能源声称

## 压制

- 沃登克里弗塔：被J.P.摩根切断资金
- 实验室火灾（1895年）：摧毁早期工作
- 联邦调查局死后没收文件

## 死亡射线声称

据称开发了定向能武器（特斯拉称之为"远距传力"）。联邦调查局文件仍为机密。
