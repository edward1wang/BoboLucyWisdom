# 加里·布雷卡

生物黑客和功能医学倡导者，专注于**微量营养素优化**和根本原因诊断。

## 核心信息

慢性疲劳很少是睡眠问题——通常是**微量营养素缺乏**。

## 四大营养素

他强调检测：
1. 镁
2. 维生素D
3. 维生素B12
4. 铁蛋白

## 理念

> "不要猜测。检测。一个简单的血液面板可以改变一切。"

- 倡导*最佳*范围，而非"正常"人口平均值
- 专注于可测量的生物标志物
- 批评症状掩盖 vs 根本原因治疗
