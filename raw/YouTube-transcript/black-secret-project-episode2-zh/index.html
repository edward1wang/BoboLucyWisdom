    关键人物（中文）¶

    • 威尔伯特·史密斯 (Wilbert Smith) — 加拿大Project Magnet负责人
    • 万尼瓦·布什 (Vannevar Bush) — MJ-12创始人
    • 奥本海默 (Oppenheimer) — 曼哈顿计划负责人，MJ-12拒绝者
    • 纳尔逊·洛克菲勒 (Nelson Rockefeller) — 1956年情报重组核心人物
    • 艾森豪威尔 (Eisenhower) — 失去控制权的总统

    UFO计划演变¶

    • 1947年9月 — Project Sign (信号计划) + MJ-12成立
    • 1948年底 — Project Grudge (怨恨计划)
    • 1952年 — Project Blue Book (蓝皮书计划) = 掩盖计划

    重要文件¶

    1. 1950年FBI给胡佛的简报 — 罗斯威尔事件，3架坠毁
    2. 1950年加拿大Wilbert Smith备忘录 — UFO机密性高于原子弹
    3. 1953年CIA心理战备忘录 — 利用UFO进行欺骗行动

    1956年关键事件¶

    • Eisenhower组建PFIAB (后更名为PFIAC)
    • Nelson Rockefeller领导重组五角大楼/CIA/DIA
    • 创造了"迷宫"般的保密系统
    • 总统和国会被系统性地隔绝

    核心技术¶

    • 标量波武器 — 特斯拉死光，击落飞碟
    • 反重力 — 逆向工程研究
    • 零点能源 — 被压制的自由能源
